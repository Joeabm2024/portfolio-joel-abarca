## Publicación en GitHub Pages

1. Crear un repositorio en GitHub.
2. Subir los archivos del proyecto.
3. Ir a Settings.
4. Entrar a Pages.
5. Seleccionar:
   - Source: Deploy from branch
   - Branch: main
   - Folder: /root
6. Guardar los cambios.
